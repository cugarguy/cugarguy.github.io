# cugarguy.github.io

Professional portfolio and personal website for Bob Steeger, Product Management Professional.

## Project Structure
- `live/`: A localized, static archive of the current live website (scraped from Blogger).
- `assets/`: Assets for the legacy/draft Bootstrap version of the site.
- `scratch/`: Utility scripts used for maintenance and data tasks.
- `.context/`: Project memory harness tracking goals, focus, and decisions.

## Live Website
The current live site is hosted at [bobsteeger.com](http://www.bobsteeger.com).
The `live/` folder contains a point-in-time archive of this site, including:
- Home Page (`index.html`)
- Online Resume (`blog-page.html`)
- All blog posts and localized assets.
